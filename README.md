## zh-address-parse

### java

-   加入依赖

```xml

    <dependency>
      <groupId>com.github.daihy8759</groupId>
      <artifactId>zh-address-parse</artifactId>
      <version>1.0.1</version>
    </dependency>
```

-   使用

```java

    import com.github.daihy8759.util.ZhAddressParser;

    ZhAddressParser.parse("", true, true, true)
```
